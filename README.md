## Installation

- Install vagrant

- Start the vm and open an ssh session

  ```
  cd /path/to/rom-vagrant
  vagrant up
  vagrant ssh
  ```

## Setup

```
export ROM_ROOT="/path/to/clones/of/rom/projects"
export ROM_VAGRANT_BOX="ubuntu-12.04"

# optional
export ROM_DOTFILES="path/to/your/dotfiles"
```
